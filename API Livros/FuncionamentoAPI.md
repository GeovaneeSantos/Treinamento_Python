# API de Livros
</br> 

### Problema inicial

- Eu estava estudando sobre progrmação e me deparei com o termo API  *(Aplication Programing Interface)*. APIs permitem que você comunique sistemas diferentes e transfira dados entre eles, para colocar esse conhecimento em prática decidi criar uma do zero usado a biblioteca Flask do Python.
  
  </br>
    
   - Essa API serve para consultar, deletar, alterar ou criar um novo registro dentro de um dicionário de dados que armazena livros (Nome do livro, nome do autor e id) e sei o Postman para fazer as requisições.

  </br>

  -  Detalhe: Eu ainda sou extremamente leigo no mundo da programação e principalmente em linguagem python, quando decidi fazer isso me deparei com termos como ENDPOINTS, METHODS, REQUEST...
  -  Não estava entendendo muito bem, mas como a gente só entende aquilo quando vivemos fui lá e apliquei. Para os sistemas se comunicarem eles precisam de uma URL e essa possui endpoints /[alguma coisa] que direciona para a API. Os dados enviados e também os recebidos por meio deste link são convertidos em formato json e então é feito o CRUD dos dados.\
  O próximo passo agora é integrar uma interface com sistema de login, uma conexão com banco de dados e tornar essa API uma solução para gerenciamento de histórico de livros.
